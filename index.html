Você pode contar com tecnologias consolidadas no mercado

HTML5 (Hypertext Markup Language, versão 5) é uma linguagem de marcação para a World Wide Web e é uma tecnologia chave da Internet. É a quinta versão da linguagem HTML.

CSS (Cascading Style Sheets) são folhas de estilo em cascata. O CSS é uma linguagem que complementa e formata o HTML (HyperText Markup Language, a Linguagem de Marcação de Hipertexto) organizando melhor as linhas e adicionando novas possibilidades ao código. Com ele, você pode modificar praticamente tudo dentro do seu layout (como as cores, background, características de fontes, margens, preenchimentos, posição, até a própria estrutura do site

bootstrap acelera o desenvolvimento de aplicações web alem de posuir uma arctetura magnifica

firebase é excelente para aplicações como e-commerce, registro e controle de usuários, cadastro de usuários e aqui com a {program(amar)} você tem todos esses serviços a sua disposição

JavaScript é uma linguagem de script orientada a objetos, multiplataforma. É uma linguagem pequena e leve. Dentro de um ambiente de host (por exemplo, um navegador web) o JavaScript pode ser ligado aos objetos deste ambiente para prover um controle programático sobre eles

o jquery foi o queridinho por muitos anos sendo substituidos por tecnologias mais recentes, mas não perdeu sua força e utilidade

O Node.js pode ser usado em inúmeros cenários. Conta com um dos maiores gerenciadores de pacotes e os recursos computacionais são consideravelmente menos exigidos ao usar a linguagem Node. A velocidade na disponibilização de um sistema para uso (deploy) e na replicação de máquinas pode crescer muito.

MongoDB é um software de banco de dados orientado a documentos livre, de código aberto e multiplataforma, escrito na linguagem C++. Classificado como um programa de banco de dados NoSQL, o MongoDB usa documentos semelhantes a JSON com esquemas.

O React é uma biblioteca que utiliza a linguagem de programação JavaScript e é uma das mais versáteis e populares bibliotecas do mundo

O Flutter é o framework móvel do Google que permite construir aplicativos Android e iOS com apenas um código
